# Content Creator Skill

기술 문서, 블로그 포스트, 스크립트 등 다양한 콘텐츠를 생성합니다.

## 사용 시점

- README 또는 문서 작성 시
- 블로그 포스트 작성 시
- 프레젠테이션 스크립트 작성 시
- 튜토리얼 작성 시

## 콘텐츠 유형

### 1. 기술 문서
- README.md
- API 문서
- 설치 가이드
- 아키텍처 문서

### 2. 블로그/튜토리얼
- 기술 블로그 포스트
- Step-by-step 튜토리얼
- How-to 가이드

### 3. 스크립트
- 발표 스크립트
- 영상 스크립트
- 데모 시나리오

### 4. 마케팅
- 제품 설명
- 릴리스 노트
- 공지사항

## 작성 원칙

### 1. 목적 중심
```
누가 읽는가? → 개발자? 비개발자?
무엇을 알고 싶어하는가?
어떤 행동을 이끌어내야 하는가?
```

### 2. 구조화
```
서론: 주제 소개, 문제 정의
본론: 상세 설명, 예시
결론: 요약, 다음 단계
```

### 3. 가독성
- 짧은 문장
- 적절한 헤딩
- 코드 예시
- 이미지/다이어그램

## 사용 방법

### Step 1: 콘텐츠 계획
```
- 목적 정의
- 대상 독자 식별
- 핵심 메시지 결정
- 구조 설계
```

### Step 2: 초안 작성
```
- outline 작성
- 각 섹션 작성
- 예시 추가
```

### Step 3: 다듬기
```
- 문장 다듬기
- 일관성 검토
- 필요시 이미지 추가
```
